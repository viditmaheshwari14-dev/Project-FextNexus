 🌐 FestNexus

FestNexus is a web-based platform that helps college event organizers discover and connect with potential sponsors efficiently. It provides a centralized database of companies that have previously sponsored events across top colleges, eliminating the need to start from scratch every time.

 🚀 Features
 
 🏫 Browse college fests and events  
 📊 Access sponsor history and details  
 🔍 Organized data for easy discovery  
 🌐 Clean and user-friendly interface  

 💡 Problem It Solves

Finding sponsors for college events is time-consuming and repetitive. FestNexus simplifies this by offering reliable, pre-collected sponsorship data, helping organizers save time and improve outreach.

 🛠️ Tech Stack

 Frontend: HTML, CSS, JavaScript  
 Backend: Node.js, Express  
 Database: MongoDB
